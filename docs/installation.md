# Installation

## Stable release

To install GrepColl, run this command in your terminal:

```sh
uv add grepcoll
```

Or if you prefer to use `pip`:

```sh
pip install grepcoll
```

## From source

The source files for GrepColl can be downloaded from the [Github repo](https://github.com/wowaster/grepcoll).

You can either clone the public repository:

```sh
git clone git://github.com/wowaster/grepcoll
```

Or download the [tarball](https://github.com/wowaster/grepcoll/tarball/master):

```sh
curl -OJL https://github.com/wowaster/grepcoll/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd grepcoll
uv pip install .
```
